# 그래프 이론

## 위상 정렬 (topology sort)
: 방향 그래프의 모든 노드를 방향성을 지키며 순서대로 나열하는 알고리즘
- 특정 노드 간의 순서가 정해져있을 때 사용

## 우선순위 큐 (priority queue)
- 구현 방법 : heapq 모듈, queue.PriorityQueue 클래스
